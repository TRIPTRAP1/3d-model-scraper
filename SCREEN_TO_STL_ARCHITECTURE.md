1.0;\n    \n    // Unproject with perspective\n    vec4 clip_space = vec4(ndc.xy, depth, 1.0);\n    vec4 world_space = view_proj_inverse * clip_space;\n    \n    // Perspective divide\n    return world_space.xyz / world_space.w;\n}\n```\n\n### Marching Cubes: Point Cloud вҶ’ Mesh\n\n```\n1. Voxelize point cloud into 3D grid\n2. For each voxel, determine if inside/outside model\n3. Where surface crosses grid cell edges:\n   - Interpolate vertex position\n   - Generate triangle faces\n4. Weld vertices, remove duplicates\n5. Result: Triangle mesh ready for printing\n```\n\n### Why This Works\n\nвң… **Captures exactly what user sees**\n- Not relying on asset extraction\n- Not accessing game memory\n- Uses standard D3D
